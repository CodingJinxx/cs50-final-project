# cs50-final-project #
CS50-2019 Final Web Project

### Setup Project in Editor ###
-- Setup Process may vary according --
1. Install venv 
`python3 -m pip install venv`
2. Setup venv
`venv venv`
3. Activate venv
`/ProjectDir/venv/Scripts/activate.bat`

### Running the Website ##
1. Set Flask Application
`set FLASK_APP=application.py`
2. Run
`flask run`
